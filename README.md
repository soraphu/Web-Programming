## 📄 Subject Information
|Topics|Description|
|:--|:--|
|HTML Semantic Tags|วิธีการใช้งาน Semantic Tags ต่างๆใน HTML เป็นโครงสร้างของหน้าเว็บ|
|CSS|ภาษาที่ใช้ตกแต่งหน้าเว็บให้สวยงาม|
|HTTP Method|Method การสื่อสารกับหน้าเว็บหรือ API โดย POST:ส่งข้อมูลไปให้, GET:ขอข้อมูล, PUT:การอัปเดตข้อมูลแบบเขียนทับ, DELETE:ลบข้อมูล, PATCH:การอัปเดตข้อมูลตามส่วนที่ระบุเท่านั้น |
|SQL|ภาษาที่ใช้สื่อสารกับดาต้าเบตที่เป็นลักษณะตาราง|
|Database|ฐานข้อมูลของเว็บไซต์ที่ใช้เก็บข้อมูลจำนวนมากอย่างเป็นระเบียบ และข้อมูลจะไม่หายไปหากเซิฟเวอร์ล้มเหลว|
|Cookies/Session|การจดจำข้อมูลผู้ใช้ใช้กับระบบล็อกอิน เพื่อจำให้ผู้ใช้ไม่ต้องกรอกรหัสใหม่ทุกครั้ง Cookies: จะเก็บภายในเครื่องของผู้ใช้, Session: จัดภายในเซิฟเวอร์|
